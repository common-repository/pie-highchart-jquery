== Installation ==

1. Open plugins/pie-highchart/highcharts-action.js
2. Update it with the values you want. Documentation here: http://www.highcharts.com/demo/pie-basic
3. Upload it to plugins folder (/wp-content/plugins/)
4. Activate the plugin
5. Use [highchart] shortcode to bring it.
